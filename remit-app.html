Juba Express Financial Remittance Mobile App


I had the opportunity to work with a local Finacial and Remittance Services company and design a remittance application. From discussion/conception to production, the project took 2 months to get off the ground to be fully functional.


Design Thinking Adobe CC Accessibility Visual Design Accessibility Information Architecture UI Design Prototyping Mobile Design

Project Overview

The goal for the organization in seeking a vendor to complete the design project was to design a modern mobile and web application that supports the broad global userbase. Juba Express mainly serves users from Africa, Europe and North America, and a user base ages between 18 - 75, so the goal was to redesign their mobile application to be not only easy to use but specifically adaptable.

Project Timeline: 12 - 16 weeks

My Role

The goal was to build a new network status application to make it easier for stakeholders to check network data points. The following were the key goals for the project:

Language
The users of this product are diverse and dozens of languages are represented. This presented some challenges, but through design thinking and user research we were able to find appropriate solutions.

Design and Security
Financial Services products and security go hand and hand. Security is more a measure than a feature, and this is certainly the case from a design standpoint. I enacted and designed features like end-to-end encryption and two-factor authentication to build trust which was only possible after understanding user needs.

Accessibility
This is particularly important and is crucial to meeting the goal of adaptability. Accessibility was not simply a checklist on a list of requirements, but rather deeply entwined in the user experience.
