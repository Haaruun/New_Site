Bell Network App UI

I often do freelance projects and contracts that allow me to learn new things and exercise my skills in different ways, and I had this opportunity to work with Bell Communications' internal Network and Data Analytics team to help design a new network status application with a modern design.


Web Design UI Development Design Thinking Adobe CC Accessibility JavaScript HTML/CSS Visual Design Accessibility Information Architecture UI Design Prototyping

Project Overview

The project was set and budgeted for 6 - 12 weeks, including development. However, through creative problem solving it was completed in 8 weeks, with no compromise in quality.

Project Timeline: 6 - 12 weeks

Highlights & Summary

Goal
The goal was to build a new network status application to make it easier for stakeholders to check network data points. I designed the following tools and features from scratch:

Network Status View
Designed a card with several features including a network status indicator, allowing the user to save time without having to click for more detail.

Filtering
The organization stressed the desire to be able to sort and filter the different data points and performance indicators.

Modern Design
After some brainstorming and design thinking workshops, we used a modern-minimalist design with some influence from material design. Typography and color were preset by corporate guidelines, but there was room for creativity with layout and composition.
